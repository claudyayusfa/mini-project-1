# Penjelasan Reservasi Kursi Bioskop
<img width="559" height="460" alt="image" src="https://github.com/user-attachments/assets/592e65a5-c70e-4169-98ea-d8de1d5d055f" />
di dalam menu reservasi kursi bioskop, terdaapat 4 menu, diantaranya adalah:
1. Membuat Reservasi Kursi: menampilkan kursi yang tersedia, lalu input kursi yang dipilih dan nama pemesan, setelah itu menampilkan kursi berhasil direservasi
2. Mengubah reservasi kursi: ini bisa dilakukan ketika kita sudah mengisi data reservasi di menu 1, dan menampilkan kursi berhasil diubah
3. Menghapus reservasi kursi: ini bisa dilakukan ketika kita sudah mengisi data reservasi di menu 1, dan menampilkan kursi berhasil dihapus
4. keluar: untuk keluar dari program tersebut. lalu untuk memulai mengisi, harus klik tombol running
